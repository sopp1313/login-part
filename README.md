🐰 토끼 카드게임 — 인증/로그인 모듈
토끼 카드게임의 회원가입 · 로그인 · 토큰 관리를 담당하는 백엔드 인증 모듈입니다.
이메일 + 비밀번호 방식의 자체 로그인을 제공하며, JWT 기반 인증으로 동작합니다.


✨ 주요 기능

 회원가입 (이메일 중복 검사, 비밀번호 정책 검증)
 로그인 (Access/Refresh 토큰 발급)
 Access Token 재발급 (Refresh Token 기반)
 로그아웃 (Refresh Token 무효화)
 내 정보 조회 (인증 필요 API 예시)
 이메일 인증 (선택, 추후 확장)
 비밀번호 재설정 (선택, 추후 확장)
